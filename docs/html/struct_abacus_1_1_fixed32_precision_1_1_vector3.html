ref Vector3  vector2,
ref Vector3  vector3,
ref Vector3  vector4,
ref Fixed32  amount,
out Vector3  result 
)
inlinestatic

Performs a Catmull-Rom interpolation using the specified positions. Features:

  • The spline passes through all of the control points.
  • The spline is C^1 continuous, meaning that there are no discontinuities in the tangent direction and magnitude.
  • The spline is not C^2 continuous. The second derivative is linearly interpolated within each segment, causing the curvature to vary linearly over the length of the segment.
